# Bank Account TypeScript Project

This project demonstrates the use of Object-Oriented Programming (OOP) in TypeScript by implementing a generic bank account system.

## Features
- Abstract class `BankAccount`
- Static property to track total accounts
- Private and readonly properties
- `SavingsAccount` with interest calculation
- `CheckingAccount` with transaction limit validation
- Fully written in TypeScript following strict compiler rules

## Technologies
- TypeScript
- Node.js
